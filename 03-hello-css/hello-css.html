Hola CSS

CSS nos permite dar formato (estilos) a los elementos HTML. También hay otra página asociada con este ejemplo.

Estilos de listas

Puedes dar estilos a las listas desordenadas con las siguientes viñetas:

  • disco
  • círculo
  • cuadrado

Y puedes enumerar las lista ordenas con lo siguiente:

  1. números decimales
  2. números romanos en minúsculas
  3. números romanos en mayúsculas
  4. letras en minúsculas
  5. letras en mayúsculas
  6. ¡y mucho más!
